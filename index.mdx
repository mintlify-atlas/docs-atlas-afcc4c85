<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get RTXPT running in your environment in minutes
  </p>
  
  <Steps>
    <Step title="Clone the repository">
      Clone the repository with all submodules recursively:
      
      ```bash
      git clone --recursive https://github.com/NVIDIA-RTX/RTXPT.git
      cd RTXPT
      ```
    </Step>
    
    <Step title="Configure with CMake">
      Generate the build files using CMake:
      
      ```bash
      cmake CMakeLists.txt -B ./build
      ```
      
      <Note>Ensure you're using Visual Studio 2022 with Windows 10 SDK version 10.0.20348.0 or later.</Note>
    </Step>
    
    <Step title="Build and run">
      Open the generated solution and build the Rtxpt project:
      
      ```bash
      # Build in Visual Studio or via command line
      cmake --build ./build --config Release
      
      # Run the sample
      ./bin/Rtxpt.exe
      ```
      
      The application will launch with the default scene. Use W/A/S/D to navigate and drag with the left mouse button to rotate the camera.
    </Step>
  </Steps>
</div>
