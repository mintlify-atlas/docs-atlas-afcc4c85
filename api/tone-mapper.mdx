The `ToneMappingPass` converts high dynamic range (HDR) path traced output to low dynamic range (LDR) for display, with support for multiple tone mapping operators, automatic exposure, and white balance.

## Overview

The tone mapper performs:
- Luminance computation and mipmap generation for auto-exposure
- Multiple tone mapping operators (ACES, Reinhard, Hable, etc.)
- Automatic exposure with configurable EV range
- White balance and color grading
- Manual exposure control via aperture/shutter/ISO

### ACES Filmic (Recommended)

The ACES tone mapper provides a filmic look with good contrast:

```hlsl
float3 toneMapAces(float3 color)
{
    // Cancel out the pre-exposure
    color *= 0.6;

    float A = 2.51;
    float B = 0.03;
    float C = 2.43;
    float D = 0.59;
    float E = 0.14;

    color = saturate((color*(A*color+B))/(color*(C*color+D)+E));
    return color;
}
```

### Reinhard

Simple Reinhard operator that maps luminance:

```hlsl
float3 toneMapReinhard(float3 color)
{
    float luminance = calcLuminance(color);
    float reinhard = luminance / (luminance + 1);
    return color * (reinhard / luminance);
}
```

### Hable Uncharted 2

Filmic tone curve used in Uncharted 2:

```hlsl
float3 toneMapHableUc2(float3 color)
{
    float exposureBias = 2.0f;
    color = applyUc2Curve(exposureBias * color);
    float whiteScale = 1 / applyUc2Curve(
        float3(gParams.whiteScale, gParams.whiteScale, gParams.whiteScale)).x;
    color = color * whiteScale;
    return color;
}

float3 applyUc2Curve(float3 color)
{
    float A = 0.22; // Shoulder Strength
    float B = 0.3;  // Linear Strength
    float C = 0.1;  // Linear Angle
    float D = 0.2;  // Toe Strength
    float E = 0.01; // Toe Numerator
    float F = 0.3;  // Toe Denominator

    color = ((color * (A*color+C*B)+D*E)/(color*(A*color+B)+D*F))-(E/F);
    return color;
}
```

## Exposure Modes

### Aperture Priority

Keeps aperture (f-number) constant, adjusts shutter speed when modifying EV:

```cpp
case ExposureMode::AperturePriority:
    // Set shutter based on EV and aperture
    m_Shutter = std::pow(2.f, m_ExposureValue) / (m_FNumber * m_FNumber);
    m_Shutter = clamp(m_Shutter, kShutterMin, kShutterMax);
    break;
```

### Shutter Priority

## Automatic Exposure

Automatic exposure adapts to scene brightness by computing average luminance:

```hlsl
if(gParams.autoExposure)
{
    float avgLuminance = gParams.avgLuminance;  // From CPU readback
    float pixelLuminance = calcLuminance(finalColor);
    
    // Apply key value and clamp to EV range
    finalColor *= clamp(
        (kExposureKey / avgLuminance),
        gParams.autoExposureLumValueMin,
        gParams.autoExposureLumValueMax
    );
}
```

## Color Transform

The color transform combines exposure compensation, manual exposure, and white balance:

```cpp
void ToneMappingPass::UpdateColorTransform()
{
    // Exposure scale due to exposure compensation 
    float exposureScale = pow(2.f, m_ExposureCompensation);
    float manualExposureScale = 1.f;

    if (!m_AutoExposure)
    {
        float normConstant = 1.f / 100.f;
        manualExposureScale = (normConstant * m_FilmSpeed) / 
            (m_Shutter * m_FNumber * m_FNumber);
    }
    m_ColorTransform = m_WhiteBalanceTransform * exposureScale * manualExposureScale;
}
```

## White Balance

White balance adjusts color temperature to match the scene illuminant:

```cpp
void ToneMappingPass::UpdateWhiteBalanceTransform()
{
    // Calculate color transform for the current white point
    m_WhiteBalanceTransform = m_WhiteBalance ?
        calculateWhiteBalanceTransformRGB_Rec709(m_WhitePoint) :
        dm::float3x3::identity();

    // Calculate source illuminant
    m_SourceWhite = inverse(m_WhiteBalanceTransform) * float3(1, 1, 1);
}
```

## Usage Example

```cpp
// Create tone mapping pass
auto toneMappingPass = std::make_shared<ToneMappingPass>(
    device,
    shaderFactory,
    commonPasses,
    framebufferFactory,
    compositeView,
    hdrTexture
);

// Configure parameters
ToneMappingParameters params;
params.toneMapOperator = ToneMapperOperator::Aces;
params.autoExposure = true;
params.exposureCompensation = 0.0f;
params.whiteBalance = true;
params.whitePoint = 6500.0f;  // D65 standard illuminant

// Each frame:
toneMappingPass->PreRender(params);
toneMappingPass->Render(commandList, compositeView, hdrTexture, true);
toneMappingPass->AdvanceFrame(deltaTime);
```
