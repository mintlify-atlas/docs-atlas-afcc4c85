## Overview

RTX Path Tracing is built on the [Donut framework](https://github.com/NVIDIAGameWorks/donut), providing a clean abstraction over DirectX 12 and Vulkan ray tracing APIs. The architecture is designed to be approachable for learning while maintaining production-quality features.

## Graphics API Backends

### DirectX 12

The primary and fully-supported backend:

- **DXR 1.1+** required for ray tracing
- **Agility SDK** integration for latest features
  - DirectX native Shader Execution Reordering (SM 6.9)
  - DirectX native Opacity Micromaps
- **NVAPI integration** for extended NVIDIA features

```cpp
// DirectX 12 initialization in DeviceManager
auto device = GetDevice();
if (device->queryFeatureSupport(nvrhi::Feature::RayTracingOpacityMicromap))
    m_ommBaker = std::make_shared<OmmBaker>(device, ...);
```

<Info>
DirectX 12 Agility SDK version 717-preview enables native support for Shader Execution Reordering and Opacity Micromaps. A preview driver is required for testing on NVIDIA hardware.
</Info>

### Vulkan

Vulkan support requires manual setup:

1. Install Vulkan SDK (tested with 1.3.290.0)
2. Enable CMake variables:
   - `DONUT_WITH_VULKAN=ON`
   - `NVRHI_WITH_VULKAN=ON`
3. Launch with `--vk` command line parameter

<Note>
**Current Limitations:**
- SER and OMM support is work in progress
- Debug layer may show warnings (fixes in progress)
- AMD GPU TDR issues with null TLAS instances
</Note>

### Pipeline Variants

The path tracer compiles multiple shader variants:

```cpp
bool Sample::CreatePTPipeline(donut::engine::ShaderFactory& shaderFactory)
{
    // Reference mode - all bounces, accumulation
    m_ptPipelineReference = m_ptPipelineBaker->CreateVariant(
        "PathTracerSample.hlsl", 
        {{ "PATH_TRACER_MODE", "PATH_TRACER_MODE_REFERENCE" }}
    );
    
    // Build stable planes - first pass for denoising
    m_ptPipelineBuildStablePlanes = m_ptPipelineBaker->CreateVariant(
        "PathTracerSample.hlsl",
        {{ "PATH_TRACER_MODE", "PATH_TRACER_MODE_BUILD_STABLE_PLANES" }}
    );
    
    // Fill stable planes - second pass for denoising  
    m_ptPipelineFillStablePlanes = m_ptPipelineBaker->CreateVariant(
        "PathTracerSample.hlsl",
        {{ "PATH_TRACER_MODE", "PATH_TRACER_MODE_FILL_STABLE_PLANES" }}
    );
}
```

### Acceleration Structure Management

```cpp
void Sample::CreateAccelStructs(nvrhi::ICommandList* commandList)
{
    // Create Bottom-Level Acceleration Structures (BLAS)
    CreateBlases(commandList);
    
    // Create Top-Level Acceleration Structure (TLAS)
    CreateTlas(commandList);
}

void Sample::BuildTLAS(nvrhi::ICommandList* commandList) const
{
    std::vector<nvrhi::rt::InstanceDesc> instances;
    
    for (const auto& instance : m_scene->GetSceneGraph()->GetMeshInstances())
    {
        const bool useOmmBLAS = m_ommBaker && 
                               m_ommBaker->UIData().Enable && 
                               hasAttachementOMM;
        
        instanceDesc.bottomLevelAS = useOmmBLAS ? 
            mesh->AccelStructOMM.Get() : mesh->accelStruct.Get();
        instances.push_back(instanceDesc);
    }
    
    commandList->buildTopLevelAccelStruct(m_topLevelAS, 
                                         instances.data(), 
                                         instances.size());
}
```

## Integrated SDKs

### RTXDI (RTX Direct Illumination)

Provides ReSTIR DI and ReSTIR GI for efficient direct and indirect lighting:

```cpp
std::unique_ptr<RtxdiPass> m_rtxdiPass;

void Sample::RtxdiSetupFrame(
    nvrhi::IFramebuffer* framebuffer,
    PathTracerCameraData cameraData,
    uint2 renderDims)
{
    m_rtxdiPass->PrepareForFrame(commandList, cameraData, renderDims);
}
```

### NRD (NVIDIA Real-time Denoiser)

Integrates ReLAX and ReBLUR denoisers with up to 3-layer path space decomposition:

```cpp
std::unique_ptr<NrdIntegration> m_nrd[cStablePlaneCount];

void Sample::Denoise(nvrhi::IFramebuffer* framebuffer)
{
    for (int i = 0; i < activeStablePlaneCount; i++)
    {
        if (m_nrd[i])
            m_nrd[i]->RunDenoiser(commandList, denoiserInputs);
    }
}
```

### OMM (Opacity Micromaps)

Accelerates alpha-tested geometry using hardware-accelerated opacity micromaps:

```cpp
std::shared_ptr<OmmBaker> m_ommBaker;

void Sample::CreateOpacityMicromaps()
{
    if (!m_ommBaker) return;
    
    m_ommBaker->BakeOpacityMicromaps(
        commandList,
        m_scene->GetSceneGraph()->GetMeshInstances()
    );
}
```

### Streamline (DLSS 4.0)

Provides NVIDIA DLSS Super Resolution, Ray Reconstruction, Frame Generation, and Multi Frame Generation:

```cpp
#if DONUT_WITH_STREAMLINE
donut::app::StreamlineInterface& streamline = 
    GetDeviceManager()->GetStreamline();

if (streamline.IsDLSSAvailable())
{
    streamline.SetDLSSOptions(dlssOptions);
    streamline.EvaluateDLSS(commandList, dlssInputs);
}
#endif
```

## Shader Execution Reordering (SER)

SER significantly improves ray tracing performance by reordering divergent shader threads:

```cpp
// DX SER API (SM 6.9)
#if USE_DX_HIT_OBJECT_EXTENSION
dx::HitObject hit = dx::HitObject::FromRayQuery(rayQuery);
dx::MaybeReorderThread(hit, coherenceHint, numCoherenceHints);
dx::HitObject::Invoke(hit, payload);
#endif

// NVAPI SER (fallback)
#if USE_NVAPI_HIT_OBJECT_EXTENSION
NvHitObject hit;
NvMakeHitWithRecordIndex(recordIndex, SceneBVH, ...);
NvReorderThread(hit, coherenceHint, numCoherenceHints);
NvInvokeHitObject(SceneBVH, hit, payload);
#endif
```

<Info>
SER can provide 10-40% performance improvement depending on scene complexity and material diversity.
</Info>

## Render Loop

```cpp
void Sample::Render(nvrhi::IFramebuffer* framebuffer)
{
    // 1. Update camera and views
    UpdateViews(framebuffer);
    
    // 2. Update lighting and environment
    UpdateLighting(commandList);
    
    // 3. Update acceleration structures (if needed)
    BuildTLAS(commandList);
    
    // 4. Path trace the scene
    PathTrace(framebuffer, constants);
    
    // 5. Denoise (real-time mode)
    if (m_ui.ActualUseStandaloneDenoiser())
        Denoise(framebuffer);
    
    // 6. Post-process (TAA/DLSS, tone mapping, bloom)
    PostProcessAA(framebuffer, reset);
    PostProcessPreToneMapping(commandList, compositeView);
    PostProcessPostToneMapping(commandList, compositeView);
}
```
